# HANA BI Website

A modern, high-performance landing page for HANA BI - a luxury aesthetic clinic combining Korean innovation and Japanese minimalism.

## Tech Stack

- **Framework:** Next.js 14+ (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Language:** TypeScript
- **Direction:** RTL (Right-to-Left) for Hebrew content

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` - Next.js App Router pages and layouts
- `components/` - Reusable React components
- `public/` - Static assets

## Design System

- **Colors:** Medical White, Charcoal Grey, Deep Red, Soft Beige
- **Typography:** Heebo (Hebrew font from Google Fonts)
- **Spacing:** Extensive whitespace for luxury feel
- **Animations:** Slow, elegant fade-ins using Framer Motion
